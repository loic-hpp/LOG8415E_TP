# LaTeX Report Template - Setup Instructions

Follow these steps to set up your environment for working with this LaTeX report template.

## 1. Install Required Software

- [MiKTeX](https://miktex.org/download): Download and install MiKTeX for LaTeX compilation.
- [Strawberry Perl](https://strawberryperl.com/): Download and install Strawberry Perl (required for some LaTeX tools).

## 2. Set Up VS Code

- Install the **LaTeX** extension in Visual Studio Code.

## 3. Update MiKTeX Dependencies

- Open the **MiKTeX Console** application.
- Go to the **Updates** tab and install all available updates to ensure all dependencies are up to date.


## 4. Build the LaTeX Project

- Open the project folder in Visual Studio Code.
- Use the **LaTeX Workshop** extension's build command (usually by clicking "Build LaTeX project" or pressing `Ctrl+Alt+B`).
- If prompted, allow the extension to install any missing LaTeX dependencies.


---

You are now ready to compile LaTeX documents using this template!
